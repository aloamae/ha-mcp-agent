Je comprends mieux maintenant. Il semble que vous avez trouvé les automatisations et les helpers suivants :

Automations :
automation.chauffage_vacances : contrôle le chauffage lorsque le mode est "vacances"
automation.chauffage_manuel : contrôle le chauffage lorsque le mode est "manuel"
automation.chaufage_planning : contrôle le chauffage en fonction du planning (pas exactement "planning", mais peut s'aggraver à cela)
automation.chauffage_humidite : contrôle le chauffage en fonction de l'humidité
Helpers :
input_boolean.mode_vacances : permet de définir le mode "vacances"
input_boolean.mode_manuel : permet de définir le mode "manuel"
input_select.mode_chauffage : permet de sélectionner l'action de chauffage
input_number.consigne_temperature : permet de configurer la consigne de température
